🔴 Icon Generator

Click each "Download" button, then save to the correct folder. See instructions below.

📋 Instructions

  1. Open this page in your browser (double-click icon-gen.html).
  2. Click Download All or download each icon individually.
  3. Create two folders next to index.html: icons/ and images/.
  4. Move downloaded files to the correct folder as shown under each icon.
  5. Host the game on GitHub Pages (free):
    • Create a GitHub repo → upload all files → Settings → Pages → Deploy from main.
    • Your URL will be: https://<username>.github.io/<repo>/
  6. Edit Package.appxmanifest — replace REPLACE_WITH_YOUR_HOSTED_URL with your GitHub Pages URL.
  7. Go to pwabuilder.com, enter your URL → Generate Package → Windows → set the Package Identity fields (already in the manifest) → Download MSIX.
  8. Submit the MSIX in Microsoft Partner Center.
